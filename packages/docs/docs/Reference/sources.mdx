## Source Resource {#source-resource}

```ts
export interface SourceResource {
  id: string;
  type: 'source';
  attributes: {
    amount: number;
    billing?: Billing;
    currency: Currency;
    livemode: boolean;
    redirect: SourceRedirect;
    status: SourceStatus;
    type: SourceType;
    created_at: number;
    updated_at: number;
  };
}
```

## Create Source

#### paymongo.source.create([CreateSourceParams](#create-params)): [SourceResource](#source-resource)

```js title="Example"
paymongo.source.create({
  data: {
    attributes: {
      amount: 10000,
      currency: 'PHP',
      redirect: {
        failed: 'https://example.com',
        success: 'https://example.com',
      },
      type: 'gcash',
    },
  },
});
```
