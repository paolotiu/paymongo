# Payment Method

[Official Docs](https://developers.paymongo.com/reference#the-payment-method-object)

## Payment Method Resource {#payment-method}

```ts
export interface PaymentMethodResource<Metadata = MetadataType> {
  id: string;
  type: 'payment_method';
  attributes: {
    livemode: boolean;
    type: 'card';
    details: {
      last4: string;
      exp_month: number;
      exp_year: number;
    };
    billing?: Billing;
    metadata: Metadata;
  };
}
```
